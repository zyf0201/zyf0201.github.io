> [Codespaces](https://docs.github.com/cn/codespaces/about-codespaces) 是一个可配置的在线开发环境，由 GitHub 托管并由 Visual Studio Code 支持，允许您完全在云端开发。

通过codespace，可以直接在浏览器里面打开vscode，同时通过"https://github.dev/[用户名]/[仓库名]"就可以打开对应仓库
